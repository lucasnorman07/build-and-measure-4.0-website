Release version 4.0.0

Welcome to version 4.0.0 of Build and Measure! Here are some of the key highlights of this update:

  • Completely rewritten code
  • Much more stable than versions 1, 2 and 3
  • Now a desktop app
  • Works with views and viewports
  • Export to svg and pdf
  • Improved UI
  • New logo
  • More features for exact drawing
  • The spline tool
  • New commands like flip, move to front/back and split
  • More drawing/object properties and settings

Release version 3.0

The version 3.0 is the last web version of Build and Measure and is available at buildandmeasure.netlify.app.

Release version 2.0

The version 2.0 of Build and Measure is no longer available and has been replaced with version 3.0.

Release version 1.0

The version 1.0 of Build and Measure is no longer available and has been replaced with versions 2.0 and 3.0.
