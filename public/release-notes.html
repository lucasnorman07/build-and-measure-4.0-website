Release version 4.5.0

Here are the key features of this update:

  • Two new buttons for controlling the crop region
    • Select objects in crop region
    • Fit crop region to selection
  • New settings for the crop region:
    • Crop region fit margin
    • Fit crop region to grid
    • Show crop region on export
  • New keyboard shortcuts for the crop region and split command
  • New and updated tool and command icons
  • A new way to display selected objects
  • Unused images will now get purged to reduce project size
  • New save-status icons, including new loading icon
  • Imported SVG images will now show in exported PDFs
  • Improved tips/info in the editor
  • ...

Release version 4.4.0

Here are the key features of this update:

  • The drawing palette and context menu have now been replaced with a brand new "properties panel"
  • The view and sheet tabs have recieved a UI refresh
  • The "line join" and "line cap" settings are now properties on objects instead
  • There is now an option to choose how text should be embedded when exported, or if it should be converted to paths instead
  • The code for the command box has been improved and some small issues have been resolved (like when drawing images with shift)
  • Some point and center move actions have been updated (like moving the side center points on Dimension objects)
  • There is now an opacity property for both stroke and fill
  • There is now a quick way to edit text by using the "control + e" shortcut
  • Many issues fixed...
